# Surma Altai Catalog

Каталог земельных участков Surma Altai — внутренняя система для команды продаж, работающая с участками в Республике Алтай и соседних регионах. Демоверсия доступна по адресу https://surma.su/ (проект на стадии беты).

## Назначение и ключевые функции
- Поиск и фильтрация участков по кадастру, региону, площади и цене.
- Просмотр карточки с техническими данными, заметками менеджеров, ссылкой на Авито.
- Сохранение собственных описаний и ссылок по каждому кадастровому номеру (`/notes`).
- Кнопка генерации текстов через YandexGPT (`/ai/describe`) при наличии ключа.
- Карта участка открывается на отдельной странице (`map.html`) с загрузкой геометрии из публичного API StarLine Maps (используем MapLibre слой, не Яндекс).

## Содержимое репозитория
- `README.md` — описание проекта, назначение и структура.
- `AGENTS.md` — руководящие указания для контрибьюторов и операторов.
- `sample/` — обрезанный пример backend + frontend без приватных данных:
  - `backend/server.py` — HTTP сервер (ThreadingHTTPServer + статика);
  - `backend/public/` — HTML/JS/CSS с фильтрами и таблицей;
  - `backend/data/notes.json` — пример заметки;
  - `public/all_regions.csv|json` — сокращённый набор участков.

## Быстрый запуск примера
```bash
cd sample/backend
python3 server.py
```
Откройте http://localhost:8080/

## Что есть в приложении (кратко)
| Компонент | Описание |
|-----------|-----------|
| Фильтры   | Район, кадастр, артикул, площадь, цена, кнопка сброса |
| Таблица   | Список участков с пагинацией, выбор открывает панель деталей |
| Детали    | Вся информация по лоту: площади, цены, ВРИ, заметки, ссылки |
| Заметки   | POST `/notes` сохраняет текст и ссылку per кадастр |
| GPT       | POST `/ai/describe` генерирует описание по данным карточки |
| Карта     | `map.html` запрашивает координаты и рисует полигон в Leaflet |

## Продакшен-инфраструктура (кратко)
- user-level systemd unit: `surma_altai.service` (путь `~/.config/systemd/user/...`).
- nginx на `surma.su` проксирует все запросы на `127.0.0.1:8080` (HTTP only).
- YandexGPT активируется переменными из `gpt5-pro.env` (`YANDEX_GPT_API_KEY`, `YANDEX_GPT_MODEL_URI`, `YANDEX_GPT_FOLDER_ID`).
- Map API запрашивается напрямую из браузера; для продакшена важна география IP (нужно РФ).
